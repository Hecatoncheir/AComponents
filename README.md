# AComponents
Components for Angular

For install use pub:
```
dependencies:
  acomponents:
    git:
      url: https://github.com/Rasarts/AComponents.git
      path: component_name
```

For run tests for component:
```
pub run angular_test --test-arg=--tags=aot --test-arg=--platform=dartium  --test-arg=--reporter=expanded
```
